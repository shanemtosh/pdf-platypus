# PDF Platypus

PDF Platypus is a 100% client-side PDF editor focused on privacy and simplicity.

## Zero Data Retention
All PDF processing happens directly in your browser. Your files never leave your computer, and no data is ever uploaded to a server. You can even use the application while offline.

## Features
- Merge: Combine multiple PDFs into one.
- Split: Extract specific pages or ranges.
- Organize: Visually reorder pages via drag-and-drop.
- Convert: Export PDF pages as PNG or JPEG images.
- Metadata: Edit title, author, and other document properties.
- Enhance: Add watermarks, page numbers, and compress file size.

## Technology
- React 19
- TypeScript
- Vite
- pdf-lib
- pdfjs-dist
